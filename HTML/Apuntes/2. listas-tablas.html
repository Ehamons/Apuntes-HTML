Aprendiendo listas y tablas con HTML

Listas con HTML

Listas ordenadas

Las listas ordenadas son elementos que muestran los elementos en un orden. La etiqueta para listas es ol, para agregar elementos a esta lista se utiliza el comando li

Ejemplo

  1. Elemento uno
    1. Sub-elemento uno
    2. Sub-elemento dos
    3. Sub-elemento tres
  2. Elemento dos
  3. Elemento tres
  4. Elemento cuatro

Listas desordenadas

Las listas desordenadas se utilizan para mostrar o listar elementos sin un orden. Para estas listas se utiliza la etiqueta ul. Para agregar elementos se utiliza li

Ejemplo

  • Elemento uno
  • Elemento dos
  • Elemento tres
  • Elemento uno
  • Elemento dos
  • Elemento tres
  • Elemento uno
  • Elemento dos
  • Elemento tres
